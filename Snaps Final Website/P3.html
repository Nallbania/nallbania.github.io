Takeaways

While designing our wireframe prototype, based on user feedback, we added a "go back" button and a continuous home button so users would have multiple escape routes at each page. Furthermore, based on the feedback, we adjusted the mood tracker and the send a future message to yourself features to be more intuitive.
A few other changes we noted based on user testing were:
  • Make the notebook icon more obviously clickable.
  • Get rid of the journal icon at the bottom of the welcome page and replaced it with the home button.
  • Add guiding messages when the user first opens the home screen that tells them what each feature does
